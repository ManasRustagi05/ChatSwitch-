🧩 STEP 1: Create Telegram Bot
Open Telegram
Search: @BotFather
Send:

/start
Then:

/newbot
Give:
Name → Home Automation Bot
Username → must end with bot
Example: home_auto_123_bot
✅ You’ll get TOKEN
Example:123456789:ABC-XYZ...
👉 This is your BOT TOKEN

🧩 STEP 2: Start your bot
Search your bot (like):

@your_bot_name
Send:/start

🧩 STEP 3: Get CHAT ID
Open browser:

https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
You already got

🧩 STEP 4: Add Commands (Professional Touch)
Send to @BotFather:

/setcommands
Then choose your bot → paste:

start - Start bot
on - Turn ON LED
off - Turn OFF LED
status - Check status
